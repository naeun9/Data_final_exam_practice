📝 1회차 모의고사 (기본 개념 & 트랜스포머/GAN/DDPM)

1. 트랜스포머(Transformer) 모델의 어텐션 메커니즘에서 쿼리(Query, Q), 키(Key, K), 밸류(Value, V)의 역할에 대한 설명으로 가장 적절한 것은?
  • 1. Query는 입력 단어의 위치 정보를, Key와 Value는 단어의 의미 정보를 담는다.
  • 2. Query와 Key의 내적을 통해 연관성(어텐션 스코어)을 구한 뒤, 이를 바탕으로 Value들을 조합하여 문맥을 파악한다.
  • 3. Key는 찾고자 하는 목표값이며, Query는 데이터베이스에 저장된 실제 데이터 값이다.
  • 4. Value는 다음 단어를 예측하기 위한 정답(Ground Truth) 레이블 역할을 한다.
정답: 2
해설: 어텐션 메커니즘은 내가 찾고자 하는 쿼리(Q)와 데이터의 키(K)의 유사도를 계산하고, 그 가중치에 따라 밸류(V)를 가져오는 원리입니다.
2. 트랜스포머 모델에서 입력 임베딩에 포지셔널 인코딩(Positional Encoding)을 더해주는 가장 주요한 이유는 무엇인가요?
  • 1. 입력 데이터의 차원을 축소하여 연산 속도를 높이기 위해
  • 2. 단어 간의 유사도를 내적(Dot Product)으로 쉽게 계산하기 위해
  • 3. 모든 단어를 병렬로 한 번에 처리하므로 단어의 위치(순서) 정보를 모델에 부여하기 위해
  • 4. 문장의 전체 길이를 일정한 크기로 정규화하기 위해
정답: 3
해설: 트랜스포머는 순차적으로 연산하지 않아 순서 정보가 소실되므로, 사인/코사인 함수 등을 활용한 포지셔널 인코딩으로 각 단어의 위치 지문을 더해줍니다.
3. 적대적 생성 신경망(GAN)의 학습 과정에 대한 설명으로 틀린 것은?
  • 1. 생성자(Generator)와 판별자(Discriminator)가 서로 경쟁하며 성능을 높이는 미니맥스(Minimax) 게임 구조이다.
  • 2. 판별자는 진짜 데이터와 생성자가 만든 가짜 데이터를 구분하도록 학습한다.
  • 3. 생성자는 학습 과정에서 진짜 데이터(Real data)를 직접 입력받아 원본과 똑같이 복원하도록 학습한다.
  • 4. 손실 함수로 이진 교차 엔트로피(Binary Cross Entropy)를 주로 활용한다.
정답: 3
해설: 생성자는 절대 진짜 데이터를 직접 보지 못합니다. 오직 랜덤 노이즈(z)를 입력받아 가짜를 만들고 판별자의 피드백을 통해서만 갱신됩니다.
4. 디퓨전 모델(DDPM)의 전방 과정(Forward Process, q-process)에 대한 설명으로 가장 적절한 것은?
  • 1. 완전한 노이즈 이미지로부터 시작하여 점진적으로 노이즈를 제거해 원본 이미지를 복원한다.
  • 2. 원본 데이터에 사전에 정의된 스케줄에 따라 점진적으로 가우시안 노이즈를 추가하여 노이즈 상태로 만든다.
  • 3. 입력된 이미지를 저차원의 잠재 공간(Latent Space)으로 압축하여 저장 공간을 최적화한다.
  • 4. 분류기(Classifier)를 이용하여 생성 과정을 특정 방향으로 가이드한다.
정답: 2
해설: 전방 과정은 원본 이미지에 노이즈 스케줄에 따라 점진적으로 노이즈를 주입하여 오염시키는 과정입니다.
5. 최근 널리 쓰이는 LDM(Latent Diffusion Model)이 기존의 기본 DDPM과 구별되는 가장 큰 핵심 원리는 무엇인가요?
  • 1. 픽셀(Pixel) 공간이 아닌, 오토인코더(VAE)를 통해 축소된 잠재 공간(Latent Space) 안에서 노이즈 주입 및 제거 과정을 수행한다.
  • 2. 노이즈를 추가하는 전방 과정을 완전히 생략하여 생성 속도를 수천 배 높였다.
  • 3. 이미지 생성을 포기하고, 오직 텍스트 문장만을 생성하도록 트랜스포머 구조로 완전히 대체되었다.
  • 4. 한 스텝(One-step) 만에 가우시안 노이즈에서 완벽한 이미지를 생성해내는 수학적 증명을 적용했다.
정답: 1
해설: 픽셀 공간 전체 연산은 비용이 크므로 데이터를 VAE를 통해 압축한 뒤 잠재 공간에서 디퓨전 과정을 수행하는 것이 핵심입니다.
6. 설명 가능한 AI(XAI) 기법 중 하나인 Grad-CAM이 기존 CAM과 비교했을 때 가지는 가장 큰 장점은 무엇인가요?
  • 1. 이미지의 일부 영역을 실제로 검은색 패치로 가리면서 결과를 비교하므로 직관적이다.
  • 2. Global Average Pooling(GAP) 레이어를 끼워 넣는 등 모델의 구조를 변경할 필요 없이, 기울기(Gradient)를 이용해 바로 적용할 수 있다.
  • 3. 연산량이 10배 이상 줄어들어 실시간 객체 인식 및 분할에 바로 활용할 수 있다.
  • 4. CNN 계열 모델뿐만 아니라 의사결정 트리(Decision Tree), SVM 등 모든 알고리즘에 적용할 수 있다.
정답: 2
해설: CAM은 모델 구조 변경(GAP 레이어 삽입 등)이 필요하지만 Grad-CAM은 역전파되는 기울기를 활용하여 구조 변경 없이 적용 가능합니다.
7. XAI 기법 중 '가림 민감도(Occlusion Sensitivity)' 방식에 대한 설명으로 가장 적절한 것은?
  • 1. 입력 이미지의 특정 영역을 패치로 가렸을 때 예측 확률이 얼마나 하락하는지를 측정하여 중요도를 파악한다.
  • 2. 모델 내부의 가중치 행렬 값을 직접 분석하여 가장 활성화된 뉴런을 시각화한다.
  • 3. 이미지에 노이즈를 점진적으로 추가해가며 원본 형태가 사라지는 시점을 측정한다.
  • 4. 생성 모델이 만든 가짜 이미지가 실제 이미지와 픽셀 단위로 얼마나 차이나는지를 측정한다.
정답: 1
해설: 가렸을 때 판단 확률이 떨어지는 영역이 곧 판단을 내리는 핵심 근거라는 블랙박스 접근법입니다.
8. 메타 러닝 알고리즘 MAML에서 'Inner Loop'와 'Outer Loop'의 역할로 가장 알맞은 것은?
  • 1. Inner Loop는 메타 파라미터를 갱신하고, Outer Loop는 특정 태스크에 맞춘 초기화 값을 찾는다.
  • 2. Inner Loop는 서포트 셋을 이용해 특정 태스크에 빠르게 적응하고, Outer Loop는 쿼리 셋에서의 손실을 바탕으로 메타 파라미터를 업데이트한다.
  • 3. Inner Loop와 Outer Loop 모두 동일한 데이터를 사용하여 파라미터를 두 번 연속 업데이트한다.
  • 4. Inner Loop는 생성자를 학습하고, Outer Loop는 판별자를 학습한다.
정답: 2
해설: Inner Loop는 모의고사(Support)로 특정 지식에 적응, Outer Loop는 실전(Query)을 통해 학습하는 근본 능력(Meta Parameter) 갱신입니다.
9. 범용 인공지능(AGI)이 기존의 인공지능(ANI)과 구별되는 가장 핵심적인 특징은?
  • 1. 한 번에 처리할 수 있는 데이터의 양이 단순히 수십 배 늘어난 초거대 모델이다.
  • 2. 주어진 고정된 환경 내에서만 최적의 효율을 내도록 설계되었다.
  • 3. 인간처럼 여러 환경에서 스스로 문제를 정의하며 '학습하는 방법' 자체를 터득할 수 있다.
  • 4. 오직 수치화된 정형 데이터만 처리할 수 있어 예측의 완벽성을 보장한다.
정답: 3
해설: AGI는 새로운 환경과 규칙에서도 스스로 메타 러닝을 통해 학습 방법을 터득하고 자율적으로 진화하는 범용적 사고를 수행합니다.
10. 오토인코더(Auto-Encoder)의 인코더와 디코더 구조 및 역할에 대한 올바른 설명은?
  • 1. 인코더는 데이터 차원을 축소하여 핵심 특징을 압축하고, 디코더는 이를 바탕으로 원본 데이터를 재구성한다.
  • 2. 인코더는 노이즈로부터 가짜 데이터를 생성하고, 디코더는 그 데이터가 진짜인지 판별한다.
  • 3. 인코더는 텍스트를 입력받아 이미지를 렌더링하고, 디코더는 생성된 이미지에 캡션을 단다.
  • 4. 인코더는 고해상도 이미지를 생성하고, 디코더는 여기에 점진적으로 노이즈를 추가한다.
정답: 1
해설: 압축(인코딩) 후 복원(디코딩) 과정을 통해 주요 특징(Latent)을 학습하는 비지도 학습 아키텍처입니다.

11. 오토인코더(Auto-Encoder)의 학습 목적과 특성에 대한 설명으로 가장 적절하지 않은 것은?
  • 1. 인코더는 입력 데이터의 차원을 축소하여 잠재 벡터를 추출한다.
  • 2. 디코더는 압축된 잠재 벡터를 바탕으로 원본 데이터와 최대한 동일하게 복원한다.
  • 3. 주로 지도 학습(Supervised Learning) 방식으로, 사람이 직접 라벨링한 정답 데이터가 대량으로 필요하다.
  • 4. 디퓨전 모델 중 LDM은 이 구조(VAE)를 활용하여 데이터를 압축한 후 연산을 수행한다.
정답: 3
해설: 오토인코더는 입력 데이터를 정답으로 삼아 복원하는 자가지도(비지도) 학습이므로 별도의 라벨링이 필요 없습니다.
12. 트랜스포머 모델에서 '멀티 헤드 어텐션(Multi-Head Attention)'을 사용하는 가장 주된 이유는 무엇인가요?
  • 1. 행렬 곱셈을 단순히 컴퓨터의 여러 코어로 나누어 병렬 처리 속도를 높이기 위함이다.
  • 2. 단어들 사이의 다양한 문맥적 뉘앙스를 서로 다른 여러 관점에서 동시에 학습하기 위함이다.
  • 3. 문장의 길이가 너무 길 때, 문장을 여러 조각으로 쪼개어 하나씩 순차 처리하기 위함이다.
  • 4. 어텐션 스코어가 너무 커져 기울기 폭발이 일어나는 것을 방지하기 위함이다.
정답: 2
해설: 문법, 화자, 시점 등 한 문장 안에서도 다양한 관점과 뉘앙스가 존재하므로 이를 다각도로 파악하기 위해 멀티 헤드를 사용합니다.
13. 디퓨전 모델(DDPM)의 역방향 학습 시 실제 데이터의 로그 우도를 직접 계산하지 않고, ELBO(Evidence Lower Bound)를 대안으로 사용하는 근본적인 이유는?
  • 1. 역방향 과정의 연쇄 확률 구조가 너무 복잡하여 실제 로그 우도를 직접 계산하는 것(Intractable)이 불가능하기 때문이다.
  • 2. 데이터 차원을 무조건 1차원으로 축소하여 연산 속도를 높이기 위해서이다.
  • 3. 마르코프 체인을 따르지 않고 단 한 번의 스텝으로 이미지를 생성하기 위해서이다.
  • 4. 손실 함수를 이진 교차 엔트로피(BCE)로 쉽게 변환하기 위해서이다.
정답: 1
해설: 교수님이 언급하신 대로 실제 로그 우도는 연쇄 확률로 인해 직접 계산할 수 없으므로, 유사한 추세를 보이는 ELBO(VLB)를 계산하여 최대화합니다.
14. XAI '폐쇄 민감도(Occlusion Sensitivity)' 방식에서 까만 상자(패치) 크기 설정에 대한 설명으로 옳은 것은?
  • 1. 무조건 크게 잡을수록 구체적이고 세밀한 판단 근거를 얻을 수 있다.
  • 2. 너무 크게 잡으면 어떤 특정 부위가 중요한지 구체적으로 알기 어렵고 두리뭉실해진다.
  • 3. 가장 작게(1픽셀) 잡으면 연산 속도가 가장 빠르다.
  • 4. 크기는 모델 내부 가중치 행렬 크기에 맞추어 자동으로 고정된다.
정답: 2
해설: 패치 사이즈가 너무 크면 판별 근거를 정밀하게 파악할 수 없고, 너무 작으면 연산량이 급증하므로 적절한 조절이 필요합니다.
15. MAML 방식에서 데이터를 다루는 방법에 대한 설명 중 잘못된 것은?
  • 1. 서포트 셋은 특정 태스크에 빠르게 적응하기 위해 사용하는 '모의고사' 역할을 한다.
  • 2. 쿼리 셋은 업데이트된 파라미터 성능을 평가하는 '실전 시험' 역할을 한다.
  • 3. 범용적 적응 능력을 기르기 위해 평가 시 서포트 셋과 쿼리 셋은 완전히 동일한 데이터를 사용해야 한다.
  • 4. 아주 적은 데이터(Few-shot)만으로 새로운 환경에 적응할 수 있는 초기 가중치 최적점을 찾는다.
정답: 3
해설: 서포트 셋(모의고사)과 쿼리 셋(실전)이 같으면 모델이 단순 암기를 하게 되므로 반드시 다른 데이터를 구성해야 일반화 능력을 키웁니다.
16. GAN 학습 시 판별자(Discriminator)가 진짜와 가짜 데이터를 구별할 때의 올바른 학습 방향은?
  • 1. 진짜에 대해서는 0, 가짜에 대해서는 1에 가깝게 출력하도록 갱신한다.
  • 2. 진짜에 대해서는 1(True), 가짜에 대해서는 0(False)에 가깝게 출력하도록 갱신한다.
  • 3. 진짜와 가짜 모두 0.5를 출력하게 하여 혼란을 유도한다.
  • 4. 판별자 스스로 노이즈를 추가하여 원본과 해상도를 동일하게 맞춘다.
정답: 2
